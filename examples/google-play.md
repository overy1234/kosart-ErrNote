### [2026-06-28] Google Play 광고 ID 선언 누락
- **상황**: AdMob 포함 AAB 제출 중
- **문제**: 불완전한 광고 ID 선언으로 게시 개요 제출 차단
- **원인**: Play Console 광고 ID 선언 미완료
- **시도했지만 실패한 방법**: 앱 콘텐츠 페이지에서 국가 행 클릭, 상태 아이콘 클릭
- **해결**: 광고 ID 사용 `예`, 목적 `광고 또는 마케팅` 선택 후 저장
- **검증**: 게시 개요에서 차단 문제 사라지고 제출 버튼 활성화 확인
- **재사용 조건**: AdMob SDK 포함 앱을 Play Console에 제출할 때마다
- **재발 방지**: AAB 업로드 전 광고 ID 선언 먼저 완료
- **태그**: #google-play #admob #ad-id
