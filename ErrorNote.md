# ErrorNote

> AI 작업 중 발생한 에러 및 문제점을 기록하고, 재발 방지와 최종 검토에 활용하는 참고 노트입니다.

---

## 사용 규칙

- 작업 중 에러나 문제가 발생하면 즉시 아래 형식으로 기록한다.
- 새 작업 시작 전, 이 파일을 읽고 유사한 문제가 없는지 확인한다.
- 작업 완료 후 최종 검토 시 이 파일의 항목들을 체크리스트로 활용한다.
- 해결된 항목은 삭제하지 않고 `[해결됨]` 태그를 붙여 유지한다.

---

## 에러 기록 형식

```
### [날짜] 제목
- **상황**: 어떤 작업 중 발생했는지
- **문제**: 구체적으로 어떤 에러/문제였는지
- **원인**: 왜 발생했는지 (파악된 경우)
- **해결**: 어떻게 해결했는지
- **재발 방지**: 다음에 같은 실수를 하지 않으려면
- **태그**: #카테고리
```
